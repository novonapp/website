## v1.0.0 (Stable)
* Initial Stable Release for Android, Windows, and Linux.
* Full deployment of the QuickJS-based Extension System.
* Implementation of the unified design language and typography engine.

## Alpha 0.5.x
* **0.5.2:** Resolved Cloudflare challenge loops on Android 12 through specialized WebView browser hooks. Added physical volume rocker navigation for E-ink device support.
* **0.5.1:** Integrated AniList and MyAnimeList tracking support.
* **0.5.0:** Migrated the local persistence layer from Hive to Drift (SQLite) for high-performance library and chapter content queries.

## MVP 0.1.0
* Initial prototype featuring basic HTML parsing and a minimal reading interface.
* Hardcoded source testing phase.
